``GETUID`` - User ID function
*****************************

.. index:: GETUID

.. index:: system, user ID

.. index:: user id

:samp:`{Description}:`
  Returns the numerical user ID of the current process.

:samp:`{Standard}:`
  GNU extension

:samp:`{Class}:`
  Function

:samp:`{Syntax}:`
  ``RESULT = GETUID()``

:samp:`{Return value}:`
  The return value of ``GETUID`` is an ``INTEGER`` of the default
  kind.

:samp:`{Example}:`
  See ``GETPID`` for an example.

:samp:`{See also}:`
  GETPID, 
  GETLOG
