GETGID - Group ID function
**************************

.. index:: GETGID

.. index:: system, group ID

:samp:`{Description}:`
  Returns the numerical group ID of the current process.

:samp:`{Standard}:`
  GNU extension

:samp:`{Class}:`
  Function

:samp:`{Syntax}:`
  ``RESULT = GETGID()``

:samp:`{Return value}:`
  The return value of ``GETGID`` is an ``INTEGER`` of the default
  kind.

:samp:`{Example}:`
  See ``GETPID`` for an example.

:samp:`{See also}:`
  GETPID, 
  GETUID
