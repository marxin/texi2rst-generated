Overview
--------

Functions contained in ``libiberty`` can be divided into three general categories.

.. toctree::

  Providing functions which don't exist
                                   on older operating systems. <supplemental-functions>

  These functions are sometimes buggy or
                                   unpredictable on some operating systems. <replacement-functions>

  Functions which provide useful extensions
                                   or safety wrappers around existing code. <extensions>
