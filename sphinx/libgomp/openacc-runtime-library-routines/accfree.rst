acc_free -- Free device memory.
*******************************

Description
  Free previously allocated device memory at the device address ``a``.

:samp:`{C/C++}:`

  ============  ========================
  *Prototype*:  ``acc_free(d_void *a);``
  ============  ========================

:samp:`{Reference}:`
  `OpenACC specification v2.6 <https://www.openacc.org>`_, section
  3.2.19.
